Politique des congés payés

Les employés ont droit à 25 jours ouvrables de congés payés par an.
Les congés doivent être demandés au moins 2 semaines à l'avance.
Les demandes de congé doivent être soumises via le système RH interne.

Congés exceptionnels
- Mariage : 5 jours
- Décès d'un proche : 3 jours
- Naissance : 3 jours

Procédure de demande
1. Connectez-vous au portail RH.
2. Remplissez le formulaire de demande de congé.
3. Soumettez la demande à votre responsable.
4. Attendez la validation dans les 48 heures.

En cas de refus, vous pouvez soumettre une nouvelle demande avec une autre date.
Les congés non utilisés en fin d'année peuvent être reportés sur l'année suivante dans la limite de 10 jours.
